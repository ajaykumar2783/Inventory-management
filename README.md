## 🎯 Overview

A **modern full-stack Inventory Management System** designed to manage materials, suppliers, stock, and reports with a **clean dashboard UI**.

Built like a **real SaaS product**, not just a project.

---

## ✨ Features

### 📊 Dashboard
- Real-time statistics  
- Chart-based analytics  
- Inventory insights  

### 📦 Inventory Management
- Add / Edit / Delete materials  
- Track quantity & pricing  
- Low stock alerts  

### 👥 Supplier Management
- CRUD operations  
- Search + Pagination  
- Modal-based UI  

### 📦 Stock Operations
- Stock In / Stock Out  
- Movement tracking  

### 📊 Reports
- Monthly reports  
- Export (CSV / Excel / PDF)  

### 🔐 Authentication
- Login system  
- Role-based access  
- Secure session handling
